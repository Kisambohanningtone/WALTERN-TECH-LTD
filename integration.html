Integration Guides

Step‑by‑step instructions to integrate Waltern Tech LTD services into your applications.

💰

M‑Pesa Daraja API Integration

Follow these steps to integrate M‑Pesa Daraja API into your application.

📝

Step 1 – Register App

Sign up on Safaricom Daraja portal and get credentials.

🔑

Step 2 – Configure Keys

Add consumer key and secret to your environment variables.

⚡

Step 3 – Generate Token

Use CURL or Postman to request an access token.

📲

Step 4 – Initiate STK Push

Send payment request with phone number, amount, and callback URL.

          curl -X POST https://sandbox.safaricom.co.ke/mpesa/stkpush/v1/processrequest \
          -H "Authorization: Bearer {token}" \
          -d '{ ... }'
💻

Step 5 – Handle Callback

Capture transaction response and update your database.

🧾

POS System Integration

Connect Waltern Tech POS with your inventory and sales systems:

  • Install Waltern POS SDK.
  • Configure database connection (MySQL/PostgreSQL).
  • Enable real‑time sales tracking via API endpoints.
  • Generate daily reports automatically.
📊

Merchant Dashboard Integration

Embed Waltern Tech dashboards into your admin panel:

  • Use our REST API to fetch sales and customer analytics.
  • Authenticate with OAuth2 for secure access.
  • Render charts using Chart.js or D3.js.
